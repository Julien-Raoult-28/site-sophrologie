si ça ne va pas :

et

Option 1 : Fusionner le contenu distant avec ton dépôt local (recommandé si tu veux garder ce qui est déjà sur GitHub)

git pull origin main --allow-unrelated-histories
git push -u origin main

Cela va récupérer les fichiers distants, fusionner avec les tiens, puis pousser le tout.

Option 2 : Écraser complètement le contenu distant (⚠️ cela supprimera ce qui est sur GitHub pour mettre uniquement tes fichiers locaux)

git push -u origin main --force
